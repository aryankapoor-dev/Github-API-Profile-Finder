## 📖 About

GitHub Profile Viewer is a React application that allows users to search for GitHub profiles and view their public information in real time using the **GitHub REST API**.

This project was built to strengthen my understanding of API integration, asynchronous data fetching, React Hooks, and component-based architecture.

---

## ✨ Features

* 🔍 Search GitHub users
* 👤 Display profile information
* 🖼️ User avatar and profile details
* ⚡ Real-time data fetching
* ⚛️ Component-based React architecture
* 🚀 Fast development with Parcel

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/aryankapoor-dev/GitHub-API-Profile-Finder.git
```

Go to the project directory

```bash
cd GitHub-API-Profile-Finder
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```
